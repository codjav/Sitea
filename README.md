# 🚀 Sitea – AI Website Builder

Sitea is a full-stack AI-powered platform that allows users to generate, edit, and deploy professional websites using simple text prompts. Built with modern web technologies and integrated with AI, authentication, and payment systems, Sitea provides a seamless experience from idea to live website.

## ✨ Features

- 🤖 **AI Website Generation**  
  Generate complete websites from natural language prompts using OpenRouter API

- 🎨 **Real-Time Editing**  
  Modify generated websites instantly with a clean and responsive UI

- 💾 **Project Management**  
  Save, update, and manage multiple website projects

- 🔐 **Authentication**  
  Secure login using Firebase Google Authentication

- 💳 **Credits System (Stripe Integration)**  
  Purchase credits to generate websites using Stripe payment gateway

- 🚀 **Deployment**  
  Deploy generated websites directly from the platform

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository
```bash
git clone https://github.com/codjav/sitea.git
cd sitea

2️⃣ Setup Backend
cd server
npm install

Create a .env file in /server and add:
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret
OPENROUTER_API_KEY=your_api_key
STRIPE_SECRET_KEY=your_stripe_key

Run backend:
npm run dev

3️⃣ Setup Frontend
cd client
npm install
npm run dev


🔒 Environment Variables
Make sure to never commit:
	•	.env
	•	API keys
	•	Stripe secrets
